# csfx:core

game engine built with C++23 and based on DirectX 12. 

## Tech Stack

* **Language:** C++23
* **Graphics API:** DirectX 12
* **Build System:** CMake

## Future Plans & Goals

The main objective of **csfx:core** is to rethink how modern game engines handle data layout and multi-threading from the ground up, moving away from legacy monolithic architectures:

* Replacing traditional object-oriented hierarchies with a clean, cache-friendly data system designed specifically to maximize CPU utilization and eliminate unnecessary abstraction layers.
* Fully leveraging DirectX 12 to give the engine direct, manual control over memory allocation, GPU command queues, and resource state transitions without intermediate driver overhead.
* Utilizing the latest C++23 standards to ensure a lightweight, strict codebase that minimizes compile times and prevents memory fragmentation natively.
